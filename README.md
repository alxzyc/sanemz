# Sanemz Models

E-commerce premium de action figures, bustos, miniaturas e dioramas impressos em 3D.

## Executar

```bash
npm install
npm run dev
```

## Stack

React 19, TypeScript, Vite, TailwindCSS, Framer Motion, React Router DOM, React Hook Form, Zod, Lucide React, Swiper.js, GSAP, TanStack Query, Zustand e componentes no estilo Shadcn/UI.
